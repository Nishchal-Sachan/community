# Image Assets Guide

## Leaders

Path: `/public/images/leaders/`

Examples:

- rakesh-mahto.jpg
- neeraj.jpg
- arun.jpg

## Services

Path: `/public/images/services/`

## Hero

Path: `/public/images/hero/`

## CTA

Path: `/public/images/cta/`

## Rules

- Use kebab-case filenames
- Use optimized images (webp preferred)
- Max size: 300kb
- Always use Next.js `<Image />`
